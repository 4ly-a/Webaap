# Webaap
Create a desktop application from website using Nativefier

## Installing

```sh
curl -o webaap.sh https://raw.githubusercontent.com/4lysson-a/Webaap/main/app/Webaap.sh | sudo chmod +x webaap.sh 
```

## Usage

```sh
./webaap.sh
```
